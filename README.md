# GeminiCliSdk

An Elixir SDK for the Gemini CLI — build AI-powered applications with Google Gemini through a robust, idiomatic wrapper around the Gemini command-line interface.

## Installation

Add `gemini_cli_sdk` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:gemini_cli_sdk, "~> 0.1.0"}
  ]
end
```

Documentation can be found at [HexDocs](https://hexdocs.pm/gemini_cli_sdk).
